# Walmart-Sales-Forecasting-PowerBI
An interactive Power BI dashboard analyzing historical Walmart sales trends and forecasting model accuracy.
## Project Overview
Developed an interactive Power BI dashboard to analyze historical Walmart sales trends and evaluate forecasting model accuracy across national retail locations from 2010 to 2012. 
## Key Insights & Business Value
* Identified Predictive Gaps: Engineered a custom DAX measure to isolate **Forecast Variance**, revealing that the forecasting model over-projected demand by **1.30 billion units**.
* Seasonality Trends: Mapped actual vs. forecast monthly metrics to pinpoint peak high-volume performance windows (Q2 acceleration vs. Q4 deceleration).
* Strategic Impact: Provides supply chain leaders with actionable data insights to calibrate inventory management models, preventing overstock capital traps during key seasonal quarters.
## Tech Stack
* Tool: Power BI Desktop
* Data Modeling: DAX (Data Analysis Expressions)
* Visualizations: Line Charts, Clustered Column Charts, Filter Slicers, KPI Cards
